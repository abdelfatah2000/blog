## Description

A Blog project using Typescript, Express and Mongodb.

## dotenv file

```bash
PORT=3000
CONNECTION_STRING = mongodb://127.0.0.1:27017/blog
```

## Installation

```bash
npm install
```

## Running
```bash
npm run dev
```
